enum command_chain_type

Type of a chain between two commands

Enumerator
NONE 

Doesn't have a next command

PIPE 

| Sends output to the next command

AND 

&& Only executes if the last command was successful

OR 

|| Execute another command afterwards

BACKGROUND 

& Execute the program in background
